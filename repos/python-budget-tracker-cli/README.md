# Python Budget Tracker (CLI)

A beginner-friendly command-line application to record daily expenses, categorise spending, and export data to CSV for review.

## Why this project exists
I built this project to strengthen Python fundamentals such as functions, input validation, and file handling using a simple real-world use case.

## Features
- Add daily expenses (amount, category, note)
- Validate user input to avoid incorrect data
- Store records in CSV format
- Summarise spending by category

## Tech used
- Python
- CSV file handling
- Functions
- Input validation

## How to run (local)
1. Install Python 3
2. Clone the repository
3. Run:

```bash
python main.py
